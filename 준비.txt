package란 프로그램이라 생각한다.
npm은 nodejs를 깔 대 기본적으로 깔리는 package manager이다. 프로그램을 관리해주는.
pm2는 프로세스가 꺼지는 걸 방지, 또는 파일 수정시 다시 껏다껴서 방영해주는 manager이다.

pm2 start app.js
pm2 monit  - q 버튼 누르면 나가짐
pm2 list
pm2 stop app
pm2 start app.js --watch   //watch 옵션 왜 붙이라는지 모르겠지만 붙이래 -> watch옵션을주면 변동사항이 있을때 재시작
pm2 log
pm2 kill  //pm2로 실행된 모든 프로세스 삭제&끝내기
